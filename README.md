### 使用说明
1. 打开`app`文件夹下的`pgyer.gradle`，修改其中的`apiKey`和`userKey`为自己的蒲公英的`apiKey`和`userKey`
2. 打开`cmd`或者`terminal`，输入`gradlew assembleDebug pgyer -P pgyerDesc="pgyer gradle application"`
3. 命令执行完毕后，即可在蒲公英上看到新鲜出炉的安装包和更新说明了
